1. Deskripsi Program 
   Program ini di buat untuk menjalankan sistem operasi berjalannya bus koetaradja

2. Cara Instalasi Program
   cari directory file tersebut pada cmd atau terminal dan jalankan/compile program dengan cara
   berikut:
   javac Penumpang.java
   javac Bus.java
   javac Main.java

3. Run Program
   java Main
